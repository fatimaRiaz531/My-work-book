# Prompt spec for Spec-Driven Development chapter generation

## Purpose

Generate a scholarly chapter on Spec-Driven Development in AI-native contexts.

## Constraints

- All factual claims must include citation placeholders like [CITATION_n].
- Target word count: ~1200 words.
- Include one runnable example (code snippet) and a short verification checklist.

## Output

Markdown suitable for `book/docs/spec-driven-development.md` with in-text citation tokens and a References section.
